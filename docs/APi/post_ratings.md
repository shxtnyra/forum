# Post Rating API

## Поставить лайк посту
**Endpoint:** `POST /v1/posts/{postId}/like`

**Описание:** Добавляет лайк к посту от текущего пользователя. Если пользователь уже ставил оценку этому посту, предыдущая оценка будет отменена.

**Требования:**
- Требуется аутентификация (JWT токен)

**Query-параметры:**
- `postId` (path, long) — идентификатор поста

**Пример запроса:**
```
POST /v1/posts/123/like
Authorization: Bearer <access_token>
```

**Пример ответа (200 OK):**
```json
{}
```

**Возможные ошибки:**
- 401 Unauthorized — Пользователь не аутентифицирован
- 404 Not Found — Пост не найден
- 400 Bad Request — Нельзя оценить черновик или удаленный пост

---

## Поставить дизлайк посту
**Endpoint:** `POST /v1/posts/{postId}/dislike`

**Описание:** Добавляет дизлайк к посту от текущего пользователя. Если пользователь уже ставил оценку этому посту, предыдущая оценка будет отменена.

**Требования:**
- Требуется аутентификация (JWT токен)

**Query-параметры:**
- `postId` (path, long) — идентификатор поста

**Пример запроса:**
```
POST /v1/posts/123/dislike
Authorization: Bearer <access_token>
```

**Пример ответа (200 OK):**
```json
{}
```

**Возможные ошибки:**
- 401 Unauthorized — Пользователь не аутентифицирован
- 404 Not Found — Пост не найден
- 400 Bad Request — Нельзя оценить черновик или удаленный пост

---

## Особенности работы:
1. Один пользователь может оценить пост только один раз (либо лайк, либо дизлайк)
2. Повторный вызов того же метода отменяет предыдущую оценку
3. Вызов противоположного метода (лайк после дизлайка или наоборот) заменяет оценку
4. Нельзя оценивать:
    - Черновики
    - Удаленные или скрытые посты
